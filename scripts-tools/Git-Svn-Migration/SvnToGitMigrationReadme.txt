With TortoiseSVN or Subversion for Windows
Tutorial : https://www.atlassian.com/git/tutorials/svn-to-git-prepping-your-team-migration

1. Download Tortoise SVN 
2. Install along with CLI Tools
3. Login to remote svn-repo
4. Use bat file to clone
5. Clean the new Git repository 
The clean-git script included in svn-migration-scripts.jar turns the SVN branches into local Git branches and the SVN tags into full-fledged Git tags
